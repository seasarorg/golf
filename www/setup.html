まず最新版のGolfのZIPファイルを適当なフォルダーにダウンロードして下さい
ダウンロードしたZIPファイルを、解凍して下さい

NETBEANS 5.5の場合

NETBEANS 5.5を起動し、「ファイル」メニューの「プロジェクトを開く」を実行してください。
なお、実行には JAVA 5.0を指定する必要があります。 JAVA　6.0の場合は、MatisseのLibraryが違いますので、
Importを変更する必要があり、現在は対応していません。（将来 6.0 Versionも配布予定）
自分のJava環境（1.5)を選んで下さい
次に ライブラリーを右クリックして、「プロパティ」をえらび、エラーとなっているコンパイル時ライブラリーを 消して、dist￥lib 以下のすべてのjarを違う フォルダーにCopyしたあと、Copyしたものをすべて選んでください（本来は総ては必要ではありませんが、seasar の他の 機能を使用するのに必要な場合があります）
次に、GOLF プロジェクトを右クリックして、「プロジェクトの生成物を削除して構築」を実行してください。
次に、GOLF プロジェクトを右クリックして　「プロジェクトをデバッグ」でサンプルが起動出来ます。

ECLIPSE　3.xの場合

golfディレクトリをEclipseで「ファイル→インポート→既存のプロジェクトをワークスペースへ」してインポートしてください。
「Golf」プロジェクトを右クリックして、リフレッシュした後、メインクラスを「org.seasar.golf.example.frame.ExampleLauncher」
にして実行して下さい
